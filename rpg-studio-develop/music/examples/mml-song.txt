# MML (Music Macro Language) Format
# Commands:
# T = Tempo (T120 = 120 BPM)
# O = Octave (O4 = octave 4)
# L = Default note length (L4 = quarter note)
# V = Volume (V15 = max volume)
# < = Octave down
# > = Octave up
# R = Rest
# C D E F G A B = Notes
# # or + = Sharp
# - = Flat
# Numbers after notes = length (4=quarter, 8=eighth, 16=sixteenth, 2=half, 1=whole)

# Example 1: Simple Scale
T120 O4 L4 C D E F G A B O5 C
