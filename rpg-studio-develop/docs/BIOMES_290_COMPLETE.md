## Usage in Code

```typescript
import { BIOMES_290, getBiome, getBiomesByCategory } from '@/lib/world/biomes290';

// Get specific biome
const forest = getBiome('forest_temperate');

// Get all wilderness biomes
const wilderness = getBiomesByCategory('Wilderness');

// Get all forest biomes
const forests = getBiomesByTag('forest');
```

## Tileset Generation

Each biome should have a corresponding 12×12 tileset following the master prompt:

```
12×12 isometric biome master tileset, [BIOME_NAME], Dungeons & Dragons 5e world style, 
dark-fantasy hand-painted realism, ultra-high detail, 3D isometric perspective, 
perfect 12×12 grid alignment, seamless modular edges, clean transparent background, 
256×256 tile resolution, game-ready RPG tileset
```

## Minimap Colors

Each biome has a unique minimap color for world map visualization. Colors are chosen to be:
- Visually distinct
- Thematically appropriate
- Colorblind-friendly where possible
