Audio Library Install Map
=========================

Use this map to place prebuilt audio packs into a project.

Music Packs
-----------

- `music/8bit-adventure-core/pack.json` -> copy listed files to `assets/music/`
- `music/8bit-battle-core/pack.json` -> copy listed files to `assets/music/`
- `music/8bit-town-core/pack.json` -> copy listed files to `assets/music/`

SFX Packs
---------

- `sfx/ui-core/pack.json` -> copy listed files to `assets/sounds/`
- `sfx/combat-core/pack.json` -> copy listed files to `assets/sounds/`
- `sfx/world-core/pack.json` -> copy listed files to `assets/sounds/`

Recommended Naming
------------------

- Music: `bgm_<context>.uge`
- SFX: `sfx_<action>.wav`

This keeps event pickers and team workflows consistent.
