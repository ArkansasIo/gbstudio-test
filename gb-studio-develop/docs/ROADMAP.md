# Roadmap

## Near Term

- Stabilize cross-platform packaging workflows.
- Improve startup reliability and project open diagnostics.
- Expand documentation coverage for contributor onboarding.

## Mid Term

- Improve large-project performance in editor workflows.
- Strengthen automated regression tests for build/export paths.
- Improve plugin/tooling extension discoverability.

## Long Term

- Deeper quality-of-life tooling for content-heavy projects.
- Enhanced diagnostics and recovery tooling for corrupted assets.
- Broader template ecosystem and guided onboarding flows.
