# RPG Game Maker UI System - Quick Reference

## UI Layout
- Menu Bar: File | Edit | View | Tools | Build | Run | Help
- Toolbar: New, Open, Save, Undo, Redo, Playtest, Build, Import Asset
- Panels: Project Explorer, Main Workspace, Properties/Inspector, Output/Console

## Main Features
- Map Editor (tile-based)
- Event Editor (visual scripting)
- Database Editor (items, skills, etc.)
- Asset Manager
- Playtest/Build/Export

## Supported Game Types
- Platformer, RPG, Puzzle, Shooter, Adventure, Racing

## Blueprint/Visual Scripting
- Node-based event system
- Triggers, actions, variables, conditions

## Tools for 8/16/32/64-bit Dev
- 8-bit: RGBDS, BGB, GBTD, GBMB, hUGETracker, Enchantment Game Engine
- 16-bit: WLA-DX, bsnes, SGDK, YY-CHR
- 32-bit: devkitPro, mGBA, No$GBA, PSY-Q SDK
- 64-bit: libdragon, Project64, PCSX2, PS2SDK

## Example Event
- Treasure Chest: On Interact → Play animation, Add item, Show message
